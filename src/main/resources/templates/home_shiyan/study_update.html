实验介绍

本项目将使用 Bootstrap 框架制作餐饮页面。主要实验内容是完成一个响应 式的餐饮类电商网站。

实验目的

本次实验的目的是让大家熟悉前端框架 Bootstrap。使用 Bootstrap 框架制作 前端网页，主要内容包括 Bootstrap 栅格系统原理、 CSS 组件、JavaScript 插件等， 通过学习与实践掌握快速制作适合各种终端的精美网页。

实验原理

Bootstrap，来自 Twitter，是目前最受欢迎的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的，它简洁灵活，使得 Web 开发更加快捷。为开发人员创建接口提供了一个简洁统一的解决方案。包含了功能强大的内置组件，易于定制，还提供了基于 Web 的定制。并且它是是开源的。

实验内容

1. 实现响应式布局 2. 完成网页 header 部分 3. 完成网页搜索模块 4. 完成热卖商品模块 5. 完成特色推荐模块 6. 完成轮播广告模块 7. 完成 footer 部分 8. 整合所有模块
